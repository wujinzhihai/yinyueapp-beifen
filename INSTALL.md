# 📥 HarmonyOS音乐播放器 - 安装指南

> 让鸿蒙手机用户一键安装使用

## 📱 系统要求

- **操作系统**: HarmonyOS NEXT (API 12+)
- **设备类型**: 华为/荣耀鸿蒙手机
- **网络环境**: 需要联网使用

---

## 🚀 快速安装(推荐)

### 步骤1: 下载安装包

**下载HAP文件:**
- 方式1: [GitHub Releases下载](https://github.com/wujinzhihai/yinyueapp-beifen/releases)
- 方式2: [网盘下载](#网盘下载链接)

**文件名**: `yinyue-app-v1.0.0.hap`  
**文件大小**: 约5-10MB

### 步骤2: 安装应用

1. **在手机上找到下载的HAP文件**
   ```
   打开文件管理器 → 下载 → yinyue-app-v1.0.0.hap
   ```

2. **点击HAP文件**
   - 系统会自动识别为应用安装包
   - 点击"安装"按钮

3. **等待安装完成**
   - 安装过程约5-10秒
   - 看到"安装成功"提示

4. **打开应用**
   ```
   桌面 → 音乐播放器图标 → 点击打开
   ```

---

## 🎵 首次使用

### 1. 搜索音乐
```
1. 打开应用
2. 在搜索框输入歌曲名或歌手名
3. 点击搜索
```

### 2. 播放音乐
```
1. 从搜索结果中选择歌曲
2. 点击即可播放
3. 支持播放/暂停、上一曲/下一曲
```

### 3. 查看歌词
```
1. 在播放页面
2. 点击"📝 歌词"按钮
3. 歌词会自动滚动并高亮
```

### 4. 切换音质
```
1. 点击右上角音质按钮
2. 选择: 标准/较高/高品/无损
```

## 🔧 其他安装方式

### 方式2: 通过电脑安装

**适用场景**: 电脑上已安装DevEco Studio

**步骤**:
1. 手机开启USB调试
   ```
   设置 → 关于手机 → 版本号(连点7次开启开发者模式)
   设置 → 系统和更新 → 开发者选项 → USB调试(开启)
   ```

2. 使用数据线连接手机和电脑

3. 在电脑上执行命令
   ```bash
   hdc install yinyue-app-v1.0.0.hap
   ```

4. 等待安装完成

### 方式3: 开发者安装

**适用场景**: 你是开发者,想要自己编译

**步骤**:
```bash
# 1. 克隆项目

cd yinyueapp-beifen

# 2. 使用DevEco Studio打开项目

# 3. 连接鸿蒙手机

# 4. 点击运行按钮 ▶️
```

## ❓ 常见问题

### Q1: 提示"未知来源应用"无法安装

**解决方法**:
```
设置 → 安全 → 更多安全设置 → 
安装外部来源应用 → 文件管理器 → 允许
```

### Q2: 提示"签名验证失败"

**原因**: HAP文件签名问题

**解决方法**:
1. 确认下载的HAP文件完整
2. 重新下载HAP文件
3. 联系开发者获取正确签名的版本

### Q3: 安装后无法打开

**解决方法**:
1. 检查系统版本是否为HarmonyOS NEXT
2. 检查是否有网络连接
3. 卸载重新安装
4. 查看GitHub Issues反馈问题

### Q4: 搜索不到歌曲

**原因**: 网络问题或API限制

**解决方法**:
1. 检查网络连接
2. 等待片刻后重试(API有频率限制)
3. 尝试搜索其他歌曲

### Q5: 播放失败

**自动处理**: 
- 应用会自动重试1次
- 重试失败后自动播放下一曲
- 无需手动干预

**手动处理**:
- 检查网络连接
- 尝试切换音质
- 播放其他歌曲
